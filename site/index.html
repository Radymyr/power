PROJECT POWER API

API статуса света для Siri, автоматизаций и собственных приложений

Получайте актуальный статус электроснабжения по адресу: есть ли свет сейчас, когда ближайшее отключение и когда включат снова. Подходит для Siri Shortcuts, Telegram-ботов, Home Assistant и любых HTTP-интеграций.

Попробовать в Swagger Читать ReDoc OpenAPI YAML

Один endpoint

GET /api/power?city=...&street=...&house=...

Параметр house обязателен.

Что возвращает API

В объекте current_status приходят поля has_power, next_off, next_on.

Граничные случаи времени

Логика учитывает переход через 00:00, поэтому время следующего включения/отключения не теряется в полуночных сценариях.

Быстрый старт

1) Откройте Swagger. 2) Введите city, street, house. 3) Нажмите Execute.

curl "https://power-six-lemon.vercel.app/api/power?city=%D0%92%D0%B0%D1%88%20%D0%B3%D0%BE%D1%80%D0%BE%D0%B4&street=%D0%92%D0%B0%D1%88%D0%B0%20%D1%83%D0%BB%D0%B8%D1%86%D0%B0&house=1"
Документация и API работают на одном домене Vercel. Если нужен JSON-контракт, используйте /openapi.yaml.
