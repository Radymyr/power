PROJECT POWER API

Публичная документация API отключений света

Готовый лендинг для использования API в Siri/Shortcuts, мобильных клиентах и интеграциях. Здесь есть OpenAPI спецификация и два интерфейса документации: Swagger UI и ReDoc.

Открыть Swagger UI Открыть ReDoc Скачать OpenAPI YAML

Endpoint

GET /api/power?city=...&street=...&house=...

house обязателен.

Ключевая цель

Вернуть current_status с полями has_power, next_off, next_on.

Проблема 00:00 решена

Логика расчета статуса учитывает переход через полночь и корректно показывает время следующего включения/отключения.

Пример запроса

curl "https://your-api-domain.example/api/power?city=%D0%BC.%20%D0%9A%D0%B0%D0%BC%E2%80%99%D1%8F%D0%BD%D1%81%D1%8C%D0%BA%D0%B5&street=%D0%B2%D1%83%D0%BB.%20%D0%90%D1%80%D1%85%D1%96%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%BD%D0%B0&house=22"
Если API хостится отдельно (например Vercel), обновите server URL в openapi.yaml.
